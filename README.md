## 🧰 Troubleshooting

If the app does not open:

- Check that the download finished
- Extract the folder again if you used a `.zip`
- Right-click the app and choose Run as administrator
- Make sure Windows Defender did not block the file
- Close the app and open it again

If the chart looks blank:

- Check your internet connection
- Confirm the market pair is active
- Refresh the view
- Switch to another time frame

If the app cannot connect to your exchange:

- Check your API key and secret
- Make sure the key has the right permissions
- Confirm testnet or live mode matches your exchange setting
- Regenerate the keys if needed
